## 🚩 Features

- 🌐 **Instant Deployment**: Updates on every `git push`.  
- 🎨 **Customizable Themes**: Change `_config.yml` to tweak appearance.  
- 📄 **Markdown Powered**: Write in plain text—no HTML required.  
- 📋 **Navigation Ready**: Easily link multiple pages.
